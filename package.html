Protochill

A Unity Editor Package which uses LLM technologies to help prototype or develop a game or application.

The main goal is to overcome the difficulties of human-machine interactions when dealing with vast object oriented structures.

The commonly used chatbox is often not sufficient, so the package additionnaly showcases a UML graphical interface and makes sure the correct class context is given to the LLM at each step

I initiated the idea for this project, and with the support of my school and fellow students, I successfully led a small team of two other developers to bring it to completion.

The biggest issue came with Unity's life cycle : when the package writes files the Editor needs to recompile, which effectively destroys all objects, making consistency a challenge for both package developpers and users

This issue was solved by using a cache file and a solid architecture that auto saves/updates itself in the cache file
