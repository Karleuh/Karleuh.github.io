Project 2: Desktop App

A command lign application that allows you to train a home made neural network to play Connect 4. How to test quickly : Create Joueur Joueur, Create Reseau Reseau, Create MonteCarlo MC, Train Reseau MonteCarlo 100, Play Joueur Reseau. This will let you play against a neural network which had 100 games of training against a Monte carlo algorithm

The neural network can see 1 move in the future which overrides it's decisions, the rest of it's strategy is pure Overfitted non-linear mess

The architecture is a simple forward propagation model with a gradient descent
